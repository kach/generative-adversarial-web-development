Substance and Style: Generative Adversarial Web Development

Abstract: We recruited a group of undergraduate university students from a variety of institutions to write HTML or CSS for a simple mock resume, without seeing the corresponding CSS or HTML. We hypothesize that this single-blind protocol leads developers to consciously use semantic web technologies, and thus produce better software.
